##  Method — applied to each model

```
1. INTUITION  →  understand what it does and why, before any math
2. MATH       →  derive every formula from scratch
3. CODE       →  numpy only, no frameworks
4. VERIFY     →  test against hand-calculated values
5. EXPLAIN    →  able to justify every line in a meeting
```
